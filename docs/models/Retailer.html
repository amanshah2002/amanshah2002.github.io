Retailer

  • _id : String
    Auto-generated ID of the retailer
  • name : String
    The name of the retailer
  • contactNo : BigDecimal
    Contact number of the retailer
  • contactName : String
    Name of the contact person
  • creditsBought : BigDecimal
    The credits bought by the retailer
  • creditsUsed : BigDecimal
    The credits sold/used by the retailer (this will be automatically calculated by the backend, the default value is 0)
  • shops : List
    The IDs of all the shops of the retailer
